## Quick Setup Steps

### 1. Get API Key (2 min)
Visit https://openweathermap.org/api and sign up for free

### 2. Configure (1 min)
Edit `static/js/app.js` line 2:
```javascript
const OPENWEATHER_API_KEY = 'your_key_here';
```

### 3. Run (30 sec)
**Option A:** Just open `index.html`
**Option B:** Run `python -m http.server 8000`

### 4. Enjoy!
Navigate to Dashboard, Weather, Sensors, Calculator, History
