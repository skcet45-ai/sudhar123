## Interaction Patterns

### Page Navigation
1. Click sidebar link
2. Page content fades out (0.2s)
3. New page fades in (0.4s)
4. Active link highlighted in blue
5. Page title updates

### Data Updates
1. Fetch from API (every 5s)
2. Check trend vs previous value
3. Update metric card numbers
4. Update trend arrow
5. Append to chart data
6. Animate chart update (smooth)
7. Save to localStorage

### Form Submission
1. User fills form
2. Click Calculate button
3. Button shows loading state
4. POST to API
5. Receive response
6. Display result card (slide in)
7. Show efficiency badge
8. Render comparison chart

### CSV Export
1. Click Export CSV button
2. Generate CSV from localStorage
3. Create blob and download link
4. Trigger browser download
5. Filename includes timestamp
