API SPECIFICATION
───────────────────────────────────────────────────────────────────

BASE URL: http://localhost:5000

ENDPOINT 1: GET /api/sensor-data
  Response: {
    temperature: 24.5,
    humidity: 65.2,
    soil_moisture: 58.3,
    timestamp: "2025-11-10T12:34:56Z"
  }

ENDPOINT 2: POST /api/calculate-footprint
  Request: {
    crop: "Rice",
    daily_usage: 500,

  }

    total_footprint: 15000,

    ideal_usage: 12000,
    sensor_data: { ... }
  }
